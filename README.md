Titanic: Machine Learning from Disaster 
Этот проект представляет собой комплексное исследование выживаемости пассажиров Титаника. Основная цель — построение пайплайна, который объединяет разведочный анализ данных (EDA), продвинутую предобработку и сравнение ансамблевых методов с нейронными сетями.

Основные этапы исследования
EDA: Выявлено, что пол (Sex), класс билета (Pclass) и стоимость (Fare) являются решающими факторами выживания.

Feature Engineering:

Заполнение пропусков Age медианой, рассчитанной внутри групп по полу и классу.

One-Hot Encoding для категориальных признаков (Sex, Embarked, Pclass).

Нормализация численных признаков (Age, SibSp, Parch) через MinMaxScaler и стандартизация Fare.

Моделирование:

Классика: Логистическая регрессия (L1/L2/ElasticNet), Деревья решений, Random Forest.

Бустинги: Сравнение эффективности CatBoost, XGBoost и LightGBM с использованием StratifiedKFold.

Deep Learning: Построение многослойного перцептрона (MLP) на PyTorch с использованием Dropout для борьбы с переобучением.


Особенности реализации
Проект реализован в модульном стиле:

config.yaml — централизованное управление гиперпараметрами всех моделей.

utils.py — изолированная логика предобработки.

main.py — чистый цикл обучения и оценки без лишнего вывода варнингов.

MAKEFILE - запускает весь процесс, скачивает нужные библиотеки и запускае код

для запуска всего кода введите в терминале команду make all !!!

Результаты
В ходе экспериментов лучшие результаты на кросс-валидации (по метрике ROC-AUC) показали градиентные бустинги. Нейронная сеть продемонстрировала сопоставимую точность (Accuracy) при правильной настройке регуляризации.
